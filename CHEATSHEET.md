# Neovim cheatsheet for this setup


## Save and exit

```vim
:w            -- save file
w             --
<leader>w     --

:wq           -- save and exit
<leader>wq    --

:q            -- exit without saving
<leader>q     --
q             --

:q!           -- force to quit without saving
<leader>qq    --
```


## Buffer navigation

```vim
:e <file>  -- open a file in a new buffer
<leader>e  --

:ls        -- list buffers
<leader>ls --

:bnext     -- goto next buffer
<leader>f  -- 

:bprevious -- goto previous buffer
<leader>d  --
```
